# Reference Images

This directory contains design inspiration images organized by archetype.

## Design Archetypes

### wedge-trapezoidal/
Shapes that taper from base to top, with angled front face.
- Braun-influenced designs
- Classic raked speaker enclosures
- "Forward-leaning" electronics

### soft-rect/
Rectangular forms with generous corner radii.
- Modern retro (Voxalon style)
- Dieter Rams / Braun aesthetic
- Consistent radii throughout

### compound-curve/
Organic shapes with varying curves, lofted profiles.
- 1970s flip clocks (Copal, Sanyo)
- Boat-hull bottom shapes
- Different front/back profiles

### tv-box/
Deeper than wide, prominent front face bezel.
- Vintage TV form factor
- Deep enclosures for CRT-style depth
- Strong face/body distinction

### sleeve-facecap/
Open-ended sleeve with separate face/back panels.
- Braun RT 20 style
- Wood end caps
- Face plate attaches to open sleeve

### vertical-tower/
Tall vertical orientation.
- JVC pyramid radio
- Vertical speaker towers
- Stacked component look

## Using Reference Images

When designing a new shell or face plate:

1. Identify which archetype(s) the design fits
2. Study the key proportions and radii from reference images
3. Note construction method (monolithic vs assembled)
4. Extract key parameters (rake angle, corner radius ratio, etc.)

## Adding New References

Save images with descriptive names:
```
archetype/source-description-detail.jpg
```

Examples:
- `wedge-trapezoidal/braun-t3-radio-front.jpg`
- `compound-curve/sanyo-flip-clock-side-profile.jpg`
- `sleeve-facecap/braun-rt20-construction-detail.jpg`
